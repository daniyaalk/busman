# busman
Busman is a lightweight Django ERP application

# System Requirements:
* python 3.8+
* pip
* Contents of `Pipfile`

# Features
* Product details
* Sales and Puchase Invoicing
* Search & Filter for Invoices and Products
* Adding multiple members to an organization
